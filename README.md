# 📊 Indian Question Paper Leaks Dashboard

An interactive **Tableau dashboard** that analyzes question paper leak incidents across India. This project uses data visualization to uncover trends, identify affected states, and provide meaningful insights through an easy-to-use dashboard.

---

# 📌 Project Overview

Question paper leaks have affected the credibility of various examinations in India. This project analyzes reported incidents using Tableau to help understand patterns, trends, and the geographical distribution of these cases.

The dashboard enables users to explore the data through interactive visualizations and filters.

---

# 🎯 Objectives

- Analyze question paper leak incidents across India.
- Identify states with the highest number of reported incidents.
- Explore year-wise trends.
- Compare incidents across different examination categories.
- Present insights through an interactive Tableau dashboard.
